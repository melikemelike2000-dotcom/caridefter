# CariDefter

Kredi kartı, nakit ve cari (kişi) harcama/tahsilat takibi için basit, hızlı ve **çevrimdışı** çalışan bir mobil web uygulaması (PWA).

- Hızlı kayıt (tek tuş şablonlar + tuş takımı)
- Banka/kart, kategori ve kişi yönetimi
- Aylık rapor, PDF ve Excel (CSV) dışa aktarma
- Limit/bildirim kuralları
- 4 haneli giriş şifresi (yerel kilit)
- Tüm veriler cihazda saklanır (localStorage) — internet gerekmez

## Telefona kurma
Adresi telefonun tarayıcısında aç → menüden **"Ana ekrana ekle"** de. Uygulama gibi açılır.

## Teknoloji
Saf HTML + CSS + JavaScript. Derleme aracı yok. `index.html` yeter.
